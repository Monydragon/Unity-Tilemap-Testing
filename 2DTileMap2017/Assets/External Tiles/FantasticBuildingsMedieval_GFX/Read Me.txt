I. Introduction

Thank you for purchasing the Resource Pack, "Fantastic Buildings: Medieval". This Resource Pack contains a collection of ready to use buildings and houses. This Add-on Pack is compatible with the full version of "RPG MAKER VX, RPG Maker XP, and RPG Maker VX Ace" and can be used with IG Maker to create your own game.

This explains the basic instructions of how to use the materials contained in this Resource Pack.

------------------------------------------------------------

II. System Requirements

None

------------------------------------------------------------

III. About Resources in this Materials Add-on Pack

The resources in this Resource Pack are optimized for use with the full version of "RPG MAKER VX ACE" and the instructions below in "How to add the resources to your game" is written with RPG MAKER VX ACE in mind. If you are using a different product from the RPG Maker series, you'll need to manually import the resources by following the instructions in the help file of that product.

------------------------------------------------------------

IV. How to add the Resource Pack to your game

After the Resource Pack, "Fantastic Buildings: Medieval" is completely downloaded to your computer, unzip it and place the folder in your desired location. 

Options in the Setup menu:

1. READ ME FIRST:
You must click "READ ME"
- This read-me file will be displayed.

2. Open RPG Maker VX Ace:
Open RPG Maker VX Ace and the project you wish to add the Resource Pack to. Select Resource Manager -----> Graphics/Tilesets -----> Import -----> Choose the resource files and then import them.

3. Create a new tileset:
Go to the "Tilesets" tab in your database. Create a new tileset ("Change Maximum..."), and simply match the filenames and the tilenames together (City_TileB goes into slot TileB for example). Be sure to adjust the passage settings in here as well.
