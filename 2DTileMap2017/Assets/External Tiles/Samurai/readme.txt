I. Introduction

Thank you for purchasing the Materials Add-on Pack, "RPG MAKER Materials for VX: SAMURAI". 
This Materials Add-on Pack contains a new collection of Japanese-style graphics. This Add-on Pack is compatible with the full version of "RPG MAKER VX" and can be used with RPG MAKER Series (some modification required) to create your own game.
This explains the basic instructions of how to use the materials contained in this Add-on Pack. Please read carefully before starting to create your original games. 

------------------------------------------------------------

II. System Requirements

OS: Windows 2000/XP/Vista/7

*Equivalent environment to RPG MAKER VX System Requirements is recommended.
------------------------------------------------------------

III. About graphics in this Materials Add-on Pack

The graphics contained in this Materials Add-on Pack are optimized for use with the full version of "RPG MAKER VX" and the instructions below in "How to use the Setup menu" is written with RPG MAKER VX in mind. If you are using a different product from the RPG Maker series, youﾕll need to manually import the graphics by following the instructions in the help file of that product.

------------------------------------------------------------

IV. How to use the Setup menu

After the Materials Add-on Pack, "RPG MAKER Materials for VX: SAMURAI" is completely downloaded to your computer, unzip it and run "launcher.exe" to start the Setup menu. 

Options in the Setup menu:

1. READ ME FIRST: 
You must click "READ ME FIRST"
- This read-me file will be displayed. 

2. Select your destination folder:
Launcher will install the selected add-on materials into the folder/project you specify. 
- Click "Browse" to select a destination folder. If you want to copy the materials directly into your projects, click on "Where should I install to?" to list all of your projects.

3. Select the add-ons to install:
The Add-ons are grouped into the following categories: 
Faces, Battlers, Characters, System, and Animations.
- Thumbnails of each material will be displayed. Select the ones you wish to install and click "Install". The selected materials will be installed into your destination folder/project.

4. Exit
