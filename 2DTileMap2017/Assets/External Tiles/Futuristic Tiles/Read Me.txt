I. Introduction

Thank you for purchasing the Resource Pack, "Futuristic Tiles Resource Pack". This Resource Pack contains a collection of futuristic tiles. This Add-on Pack is compatible with the full version of "RPG MAKER VX, RPG Maker XP, and RPG Maker VX Ace" and can be used with IG Maker to create your own game.

This explains the basic instructions of how to use the materials contained in this Resource Pack.

------------------------------------------------------------

II. System Requirements

None

------------------------------------------------------------

III. About Resources in this Materials Add-on Pack

The resources in this Resource Pack are optimized for use with the full version of "RPG MAKER VX" and the instructions below in "How to add the resources to your game" is written with RPG MAKER VX in mind. If you are using a different product from the RPG Maker series, you'll need to manually import the resources by following the instructions in the help file of that product.

------------------------------------------------------------

IV. How to add the Resource Pack to your game

After the Resource Pack, "RPG MAKER Futuristic Tiles Resource Pack" is completely downloaded to your computer, unzip it and place the folder in your desired location. 

Options in the Setup menu:

1. READ ME FIRST:
You must click "READ ME"
- This read-me file will be displayed.

2. Open RPG Maker VX:
Open RPG Maker VX and the project you wish to add the Resource Pack to. Select Resource Manager -----> Graphics/System -----> Import -----> Choose the resource files and then import them.

3. Passability:
After the tileset is imported, you will have to manually set the passability of the tiles yourself. You can do this by clicking the Passage Settings, and giving each tile the appropriate settings.

------------------------------------------------------------

V. How to use TileE

This resource pack contains a TileE, which contains mostly decorative objects. There objects are to be put on tables using event graphics, and some are meant to be placed on a wall. There are exceptions, of course.

PLEASE NOTE: some decorative objects for tables are two tiles high or wide, and meant to be used as a whole. Using only one tile can result in the object looking cut off.
