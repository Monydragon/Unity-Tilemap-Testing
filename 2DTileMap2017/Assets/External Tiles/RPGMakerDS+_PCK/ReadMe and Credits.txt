I. Introduction

Thank you for purchasing the Resource Pack, "RPG MAKER DS+ Resource Pack". 

This Resource Pack contains a revamped RTP ported from RPG Maker DS+ with additional materials. This Add-on Pack is compatible with the full version of RPG MAKER VX and RPG Maker VXAce. It might require some modification in order to be usable with RPG Maker XP and IG Maker. You may NOT use the resources in this pack for creating games made with engines/programs other than the RPG Makers and IG Maker.

This explains the basic methods of how to use the materials contained in this Resource Pack. Please read carefully before adding these resources to your projects. 

------------------------------------------------------------

II. System Requirements

None

------------------------------------------------------------

III. About Resources in this Materials Add-on Pack

The resources in this Resource Pack are optimized for use with the full version of "RPG MAKER VX Ace" and the instructions below in "How to add the resources to your game" are written with RPG MAKER VX Ace in mind. If you are using a different product from the RPG Makers, such as Indie Game Maker, you'll need to manually import the resources by following the instructions in the help file of that product. 

------------------------------------------------------------

IV. How to add the Resource Pack to your game

After the Resource Pack, "RPG MAKER DS+ Resource Pack" is completely downloaded to your computer, unzip it and place the folder in your desired location. 

Options in the Setup menu:

1. READ ME FIRST: 

You must click "READ ME FIRST"
- This read-me file will be displayed. Some folders also have text or image files in them on where to extract the resources and how to use them.

2. Open RPG Maker VX Ace:

Open RPG Maker VX Ace and the project you wish to add the Resource Pack to. Select Tools -----> Resource Manager -----> Graphics/Tilesets ---------> Import --------> Choose the resource files and then import them. 

There are other materials included in this pack, just check their respective folders for any text files on where to put them or their folder names. There are images provided as well to set passability for specific tiles or how to use some specific graphics.

3. High Five

The Resource Pack is now in your game. Wasn't that easy?
